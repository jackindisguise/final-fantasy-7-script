About:

This was a quick project I threw together to create a flashcard deck of all the dialogue in Final Fantasy VII. The goal was simple: create a flashcard deck, study the deck, and eventually read Final Fantasy VII from beginning to end without having to use a dictionary.

As I worked on formatting the script properly from the source files, I had the idea to compile information about kanji and other things found in the script. Eventually wanted to somehow automatically scan the script and get all the vocabulary from it. And I kind of did create a tool for doing this using Mecab. However, it takes a lot of manual labor, so I haven't gone through the effort to complete that part yet.

In the meantime, I'm sharing this for posterity's sake.
