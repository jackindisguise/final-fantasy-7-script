About:

This was a quick project I threw together to create a flashcard deck of all the dialogue in Final Fantasy VII. The goal was simple:
create a flashcard deck, study the deck, and eventually read Final Fantasy VII from beginning to end without having to use a dictionary.

As I worked on formatting the script properly from the source files, I had the idea to compile information about kanji and vocabulary as well.
I did the kanji part, and I've started doing the vocabulary part. The vocabulary part unfortunately requires a lot of manual labor, so I haven't
been able to finish it yet.

In the meantime, I'm sharing this for everyone else to use.
